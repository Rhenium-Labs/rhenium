# Rhenium

Originally built for the [Brawl Stars Discord Server](https://discord.com/brawlstars), Rhenium is an AI-powered moderation bot designed to be simple, fast, and reliable. Built _by moderators, for moderators_, it focuses on reducing manual workload while keeping communities safe and easy to manage.

## Features

- **Highlights** – Get notified when messages match custom keywords or regex patterns.
- **Ban Requests** – Require higher level moderator approval for banning users.
- **Quick Actions** – Perform common moderation actions instantly using message reactions.
- **Message Reports** – Allow users and staff to easily report problematic messages.
- **Content Filtering** – Automatic content moderation powered by the OpenAI Moderations API.

## Self-Hosting

Since the public bot requires whitelist approval & has limited capacity, we've made Rhenium as easy as possible to host yourself. See the [Self-Hosting Guide](./documentation/SelfHosting.md) for setup instructions and requirements.
